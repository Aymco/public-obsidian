avec $Z$ une variable aléatoire spéciale  
  
La probabilité de dépasser un multiple de l’écart type pour une loi normale est constante :  
$P(X > σ) = 0.32$  
$P(X > 2σ) = 0.046$  
$P(X > 3σ) = 0.0027$
